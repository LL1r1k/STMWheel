# База для гоночного симмулятора с обратной связью
Проект для STM32 в STM32CubeIDE на основе USB HID. (С/C++)

## Требования
- STM32F103
- DC Motor 12V 
- BTS7960

## Использование
Открыть проект в STM32CubeIDE
